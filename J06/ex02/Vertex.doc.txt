<- Vertex ----------------------------------------------------------------------
The vertex class handles vertex.

It wait an array with the key:
	=> 'x' : X coordinates, obligatory.
	=> 'y' : Y coordinates, obligatory
	=> 'z' : X coordinates, obligatory
	=> ’w’ :  optional, take 1.0 by default.
	=> ’color’ : color of the vertex instance of the Color class (by default white)

This Class does not purpose any method. You can set the attribute verbose at True or False
for controlling the use of this class
